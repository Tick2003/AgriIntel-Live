## Reporting a Vulnerability

We take the security of AgriIntel.in seriously. If you discover a security vulnerability, please send an e-mail to security@agriintel-demo.com. All security vulnerabilities will be promptly addressed.

**Please do not report security vulnerabilities through public GitHub issues.**

### Process

1.  **Report**: Email your findings to the security contact.
2.  **Ack**: We will acknowledge receipt within 48 hours.
3.  **Investigate**: We will verify the vulnerability.
4.  **Fix**: We will release a patch.
5.  **Announce**: We will publish a security advisory.

Thank you for improving the security of AgriIntel!
